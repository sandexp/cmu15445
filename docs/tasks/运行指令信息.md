### 打包压缩指令

#### 通用指令

0. 初始化编译
```shell
$ mkdir build
$ cd build
$ cmake ..
$ make -j
```

1. 格式检查
```shell
$ make -j format
$ make -j check-lint
$ make -j check-clang-tidy
```

2. 内存泄漏检查
```shell
$ make -j hash_table_test
$ valgrind --trace-children=yes \
   --leak-check=full \
   --track-origins=yes \
   --soname-synonyms=somalloc=*jemalloc* \
   --error-exitcode=1 \
   --suppressions=../build_support/valgrind.supp \
   ./test/hash_table_test
```
